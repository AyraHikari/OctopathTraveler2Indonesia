# Octopath Traveler 2 Indonesia
Patch Bahasa Indonesia untuk Octopath Traveler 2 (Steam)

# Cara Pakai
- Copy file BahasaIndonesia_p.pak ke: `SteamLibrary\steamapps\common\Octopath_Traveler2\Octopath_Traveler2\Content\Paks`
- Buka game nya, enjoy
